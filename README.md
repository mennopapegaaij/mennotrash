# MennoTrash 🗑️

Een endless runner spel gemaakt door Menno!

## Hoe speel je het?
- Druk op **SPATIE** of **pijl omhoog** om te springen
- Spring over de rode blokken
- Hoe verder je komt, hoe sneller het gaat
- Probeer een nieuw record te zetten!

## Starten
```
python main.py
```

## Bestanden
- `main.py` — het hoofdspel
- `speler.py` — het gele vierkantje
- `instellingen.py` — alle instellingen
